    プラグイン

    hexo-toc

    h2-h6タグを抽出し目次を自動生成してくれる

    導入

    npm install hexo-toc --save

    利用

    hexo-tag-flickr

    flickrの画像表示

    導入

    npm install hexo-tag-flickr --save

    _config.yml追記

    plugins:
  - hexo-tag-flickr

flickr_api_key: [API KEY]
flickr_cache_file_path: hexo-tag-flickr-cache.json
flickr_cache_expires: 2592000000

    利用

    {% flickr photo [ID] %}

    Makefile

    便利なコマンドまとめました

    vi Makefile

    up:
    hexo server --draft
clean:
    hexo clean
    hexo generate
deploy:
    hexo deploy -g

    サーバー起動

    $ make up

    クリア

    $ make clean

    デプロイ

    $ make deploy
